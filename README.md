# Diabetes Prediction Model

A machine learning project to predict whether a patient has diabetes based on diagnostic measures.

## Dataset

* **Source:** [PIMA Indians Diabetes Database (Kaggle)](https://www.kaggle.com/uciml/pima-indians-diabetes-database)
* **Attributes:**
    * Pregnancies
    * Glucose
    * BloodPressure
    * SkinThickness
    * Insulin
    * BMI
    * DiabetesPedigreeFunction
    * Age
    * Outcome

## Technologies Used

* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Modeling:** Scikit-Learn (Logistic Regression, Decision Trees, Random Forest, etc.)
